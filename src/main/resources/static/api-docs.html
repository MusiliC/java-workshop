Shop Application API Documentation

This is a minimal Spring Boot application structured as a Modulith, following the official Spring Modulith documentation. The site demonstrates modular design, event-based communication, and API documentation generated from tests.

  • Inventory API
  • Orders API
  • View Test Report (index.html)

Inventory API

  • GET /inventory/products - List all products
  • POST /inventory/products - Create a new product

Orders API

  • GET /orders - List all orders
  • POST /orders - Create a new order

GET /inventory/products

Returns a list of all products in the inventory.

POST /inventory/products

Creates a new product in the inventory.

GET /orders

Returns a list of all orders.

POST /orders

Creates a new order.
